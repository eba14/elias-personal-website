Personal Projects:

Sliders

Status: Released (Click on the Sliders logo above to play!)

This personal project focuses on the creation of a sliding puzzle game, "Sliders". Players can choose from difficulty levels with grid sizes ranging from 3x3 (easiest) to 7x7 (hardest). The goal is to slide the numbered tiles into a specific randomized order, which is different for each game played. The numbered tiles don't need to be in a numerical sequence; the order will vary with each new game. The puzzle game offers an engaging challenge with various levels of complexity, suitable for both beginners and puzzle enthusiasts.

The game has been released as of December 2025, with more updates to come in order to improve user experience and engagement with this strategy-based puzzle game.

Jumbled Blockers

Status: Work In Progress...

"Jumbled Blockers" is a fun and brain-stimulating puzzle game where the objective is to arrange colored discs onto matching color-coded poles. Players can only move the top disc on any pole, and they can only move one disc at a time. The game allows players to choose the difficulty by selecting how many poles to sort discs from, ranging from 4 to 7 poles. The number of discs a pole can hold is also determined by the difficulty level (in a 1:1 ratio).

The objective is to sort all the discs by color in as few moves as possible while keeping track of the limited pole capacity and disc order restrictions. Each puzzle is timed, adding an extra layer of competition. The game is intended to be engaging, helping players improve their problem-solving skills while offering various levels of difficulty.

Target Release Date: Summer 2026
